一、服务端svn的配置 1、服务器端配置： 1）安装svn，yum install svn* -y 2）创建目录，做为svnserve的的工作目录，mkdir svndir 3）创建svn镜像目录，svnadmin create rep1，做为rep1的目录文件 4）进入rep1文件夹， ①：修改auths文件， 添加 [groups] ceshi = tangwei,kangkang [rep1:/] @ceshi = rw ②：修改passwd文件， 修改 [users] tangwei = 123456 ③：修改svnserve.conf文件， 修改，去掉对应文件的注释即可 5）启动服务，svnserve -d -r /xx/xx/xx/svndir
