dockerFile的使用： 1、FROM 镜像名称:版本号 2、MAINTAINER 邮箱名称 3、RUN 在需要制作的镜像中，需要运行的命令，一般是安装使用 eg: RUN yum install xxx -y 4、CMD或者EMTRYPOINT 制作的镜像被使用时需要运行的命令。eg:CMD ["/usr/bin/httpd", "-D", "FOREGROUND"] 5、EXPOSE 需要暴露的端口 eg:EXPOSE 80 6、WORKDIR 启动镜像后，运行的根目录。eg: WORKDIR /var/www/html 7、ADD 将宿主机中的文件或者目录，复制到container中，如果宿主机中是压缩文件，ADD时会自动解压，eg: ADD html /var/www/ 8、COPY 将宿主机中的文件或者目录，复制到container中 eg: COPY html /var/www 9、ENV 设置环境变量 eg: ENV MYSQL_ROOT_PASSWORD tangwei 10、VOLUME 设置镜像的文件卷挂载 eg: VOLUME ["/var/docker/data"] 运行dockerfile： docker build -t 镜像名称:版本号 ./Dockerfile
