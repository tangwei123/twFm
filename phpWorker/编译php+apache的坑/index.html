首先要知道，让你编译安装只是为了让你更熟悉，我们的php要在linux系统上跑起来，需要哪些组件更好的为以后做准备，一般情况下 我们都是通过yum install xxx来安装的。 一、先编译安装apache，如要支持https，需要编译apr apr-util, apr-iconv, pcre 1、安装apr： 1.1：解压apr-1.5.2.tar.gz这个包，命令：tar -zxvf apr-1.5.2.tar.gz 1.2：进入apr-1.5.2文件夹，cd apr-1.5.2 1.3：编译安装apr, 先运行：./configure --prefix=/usr/local/apr 预编译apr文件，指示文件安装到/usr/local/apr文件夹中。 1.4：编译安装apr，命令： make && make install 2、安装apr-util： 1.1：解压apr-util-1.5.4.tar.gz这个包，命令：tar -zxvf apr-util-1.5.4.tar.gz 1.2：进入apr-util-1.5.4文件夹，cd apr-util-1.5.4 1.3：编译安装apr-util，先运行：./configure --prefix=/usr/local/apr-util 预编译apr-util，指示文件安装到/usr/local/apr-util文件夹中。可能需要指定apr这个包的bin路径带上 --with-apr=/usr/local/apr/ 1.4：编译安装apr-util,命令：make && make install 3、安装apr-iconv：步骤类似安装apr-util。 4、安装apache： 1.1：解压httpd-2.4.23.tar.gz这个包，命令：tar -zxvf httpd-2.4.23.tar.gz 1.2：进入httpd-2.4.23文件夹，命令：cd httpd-2.4.23 1.3：编译安装httpd，先运行，./configure --prefix=/usr/local/httpd --enable-so --enable-ssl --with-ssl=/usr/bin/openssl --with-apr=/usr/local/apr --with-apr-util=/usr/local/apr-util --with-apr-iconv=/usr/local/apr-iconv 1.4：编译安装httpd：命令：make && make install 二、安装mysql: 运行命令即可：yum install mariadb* -y 二、编译安装php，必须要让apache解析php，那么要加上参数--with-apxs2=/xxxxx/xxx 1、解压安装包进入安装文件略过，这边需要安装很多类似的包，具体哪些自行baidu。 2、运行命令如下：./configure --prefix=/usr/local/php7 --exec-prefix=/usr/local/php7 --with-config-file-path=/usr/local/php7/etc --with-mysql-sock=/var/lib/mysql/mysql.sock --with-mcrypt --with-mhash --with-openssl-dir=/usr/local/opt/openssl --with-zlib --enable-zip --enable-mysqlnd
