一、下载opencv的源码包：
	wget https://github.com/opencv/opencv/archive/3.4.0.tar.gz

二、安装编译所需要的依赖项：
	yum install cmake gcc gcc-c++ gtk+-devel gimp-devel gimp-devel-tools gimp-help-browser zlib-devel libtiff-devel libjpeg-devel libpng-devel gstreamer-devel libavc1394-devel libraw1394-devel libdc1394-devel jasper-devel jasper-utils swig python libtool nasm

三、随后解压opencv的目录，进入到opencv的目录中，创建一个build（自己定义的，不是强制要求）文件夹：
	mkdir build

四、进入到build目录中：

	cd build

五、运行编译命令：（如果opencv的根目录下有CMakeCache.txt文件切记删掉） 下面命令的..是因为要出build目录，找到CMakeLists.txt这个文件。
	cmake .. -DCMAKE_BUILD_TYPE=RELEASE (安装的版本) -DCMAKE_INSTALL_PREFIX=/usr/local(这儿是安装目录)  

六、安装：（在build目录下）
	make && make install



编译完成之后，头文件已经在/usr/local/include中了！库文件已经在/usr/local/lib中了！
