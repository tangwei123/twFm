mac上一句 brew install qt@5.7就能搞定一切！！

一、新建一个文件夹叫Qt5
	mkdir Qt5

二、进入到Qt5文件夹中
	cd Qt5

三、下载Qt的源码包
	从ffmpeg.club中下载

四、编码(如遇到什么没准的 就装一下)
	./configure -prefix /usr/local/qt5 -platform macx-clang -no-framework

	这个时候他会问你：“Which edition of Qt do you want to use?”然后给你两个类型供选择：
        	Type 'c' if you want to use the Commercial Edition.
        	Type 'o' if you want to use the Open Source Edition.
        	第一个是商业版，第二个是自由版。
        	选择 'o'（Qt/嵌入式自由版是Qt为了开发自由软件提供的嵌入式版本）。
        	这时候，出现许可界面。
        	选择  'yes' 接受许可协议。

五、安装
	make & make install
